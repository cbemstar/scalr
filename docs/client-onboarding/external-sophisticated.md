## 3. Optional expectation-alignment mini-workshop (30–45 min)

Use when sales flagged politics or mixed priorities.

**Prompts:**  

- What does each stakeholder need to feel good at launch?  
- What is **not** changing in this phase?  
- What would make this project a failure for you?

**Output:** 5 bullets in the recap email under “Aligned expectations”.
